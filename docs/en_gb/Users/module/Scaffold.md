# Scaffold Module - User Guide

## Overview

Scaffold automatically places blocks beneath you as you walk. It features various bridge modes and bypass capabilities for different servers.

## Basic Usage

1. Enable the module
2. Hold blocks in your hand
3. Walk forward - blocks will be placed automatically

## Main Settings

### Bridge Mode

Choose your bridging style.

**Normal** - Standard scaffold

**WatchDog** - For Hypixel and other WatchDog anti-cheat servers
- Smooth movement
- Less detectable

**GodBridge** - GodBridge technique
- Automatic jumping
- Fast bridging

**Telly** - Telly bridge
- Places while jumping

**Keep-Y** - Maintains height while moving
- Less likely to fall

**KeepUP** - Continuously ascends

**Andromeda** - Special placement mode

### Rotation Mode

Controls camera movement.

**Normal** - Standard camera movement

**WatchDog/WatchDog2** - WatchDog-compatible camera movement

**Stabilized** - Stable camera

**Telly** - For Telly bridge

**Snap** - Quick camera switch

**Spin** - Continuous rotation

**None** - No camera movement (more detectable)

### Sprint Mode

Configure running behavior.

**Normal** - Standard running

**WatchDog** - WatchDog compatible

**Legit** - Appears more legitimate

**Air** - Sprint in air

**Ground** - Sprint only on ground

**Custom** - Custom configuration

**None** - No sprinting

### Tower

Hold jump key to tower straight up.

**Separate settings for horizontal and vertical movement:**
- Vanilla - Standard tower
- NCP - NoCheatPlus compatible
- BlocksMC - BlocksMC compatible
- AAC - AAC compatible

### Other Useful Settings

**SafeWalk** - Prevents falling (recommended)

**Eagle** - Sneak at edges
- Choose from Packet/Silent/Normal

**Swing** - Arm swing animation (visual)

**AutoBlock** - Automatically switch to blocks
- Switch - Slot switching
- Spoof - Packet spoofing

**Timer** - Game speed adjustment (for bypass)

**HitableCheck** - Placement validation
- Simple - Basic check
- Strict - Strict check
- Off - Disabled
