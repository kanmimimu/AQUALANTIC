# Scaffold モジュール - ユーザーガイド

## 概要

Scaffoldは足元に自動でブロックを置いてくれるモジュールです。様々なブリッジモードやバイパス機能があり、多くのサーバーで使用できます。

## 基本的な使い方

1. モジュールを有効化します
2. ブロックを持ちます
3. 前進するだけで自動的にブロックが配置されます

## 主な設定

### ブリッジモード

ブリッジのやり方を選択できます。

**Normal** - 通常のScaffold

**WatchDog** - Hypixel等のWatchDogアンチチート対応
- スムーズな動作
- 検知されにくい

**GodBridge** - GodBridge技術
- 自動ジャンプ
- 高速ブリッジ

**Telly** - Tellyブリッジ
- ジャンプしながら配置

**Keep-Y** - 高さを保ったまま移動
- 落下しにくい

**KeepUP** - 上昇し続ける

**Andromeda** - 特殊な配置モード

### Rotationモード

視点の動きを制御します。

**Normal** - 通常の視点移動

**WatchDog/WatchDog2** - WatchDog対応の視点移動

**Stabilized** - 安定した視点

**Telly** - Telly用

**Snap** - 素早い視点切り替え

**Spin** - 回転し続ける

**None** - 視点を動かさない（検知されやすい）

### Sprintモード

走り方を設定できます。

**Normal** - 通常の走行

**WatchDog** - WatchDog対応

**Legit** - 正規プレイに近い動作

**Air** - 空中で走る

**Ground** - 地上のみ

**Custom** - 細かくカスタマイズ

**None** - 走らない

### Tower

ジャンプキーを押すと真上にタワーできます。

**水平移動時と垂直移動時で別々に設定可能:**
- Vanilla - 通常のタワー
- NCP - NoCheatPlus対応
- BlocksMC - BlocksMC対応
- AAC - AAC対応

### その他便利な設定

**SafeWalk** - 落下防止（有効推奨）

**Eagle** - エッジでスニーク
- Packet/Silent/Normal から選択可能

**Swing** - 手を振る（見た目）

**AutoBlock** - 自動でブロックに切り替え
- Switch - スロット切り替え
- Spoof - パケットで偽装

**Timer** - ゲーム速度調整（バイパス用）

**HitableCheck** - 配置可能チェック
- Simple - 簡易
- Strict - 厳密
- Off - オフ
